table 使用說明

外觀樣式

  • 資料顯示方式 列表 or 卡片

    data-card-view="true" = 顯示方式為卡片 data-card-view="false" = 顯示方式為列表

  • table 樣式

    預設classes 為 'table table-bordered table-hover'

      可使用以下class 來調整樣式
    • 外邊框 table-bordered
    • 無內邊框 table-borderless
    • hover效果 table-hover
    • table奇偶數列不同色 table-striped
    • 暗色系 table-dark
    • 大小 table-sm

  • table 樣式調整

    調整bootstrap中_variables.scss的變數 $table-hover-bg

    $table-hover-bg: pink !default;

  • thead 樣式調整

    thead 背景與下邊框的樣式

    .bootstrap-table .fixed-table-container.fixed-height .table thead th { background: #e2f7f7; border-bottom: 1px solid #182767;}

  • tbody 樣式調整

操作優化

  • 滑鼠選取範圍

    data-click-to-select="true" : 點選row時就可以選取
    data-click-to-select="false" : 點選checkbox才能選取
